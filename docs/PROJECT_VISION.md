# Project Vision

Jade Textile Factory System V2 is a professional ERP/MES for a real garment factory.

The factory chain is:

Order -> Material Readiness -> Planning -> Cutting -> Printing -> Embroidery -> Sewing -> Finishing -> Packing -> Shipment -> Management KPIs -> Management Actions

The core product identity is the Live Factory Map.

The operating vision is:

- One Factory
- One Data Source
- One Operational Flow
- One Management View
- One Agent Supporting All Teams

The default UI language is English. Arabic and Turkish readiness is prepared through the i18n structure, but translations are not implemented yet.
